What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It is a modern software development practice that automates building, testing, and deploying applications. Continuous Integration helps developers merge code changes frequently, while Continuous Deployment makes updates available automatically. This improves speed, reliability, and reduces human errors.

My Pipeline

When I push my project files to GitHub, GitHub Actions automatically starts a workflow. The workflow checks that the required files exist. Then it uploads the website files and deploys them to GitHub Pages. This allows the website to be updated online automatically after every push.
